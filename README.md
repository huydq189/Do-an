# QUẢN LÍ NHÀ SÁCH

**Đồ án cuối kì môn nhập môn công nghệ phần mềm**

_**MÔ TẢ**_:
>Đáp ứng nhu cầu tin học góp phần tạo điều kiệu quản lý nhà sách dễ dàng hơn.

_**CHỨC NĂNG CHÍNH**_: 
+ Lập hóa đơn, xem hóa đơn
+ Quản lí sách
+ Bán sách, quản lí nợ
+ Báo cáo tồn và báo cáo nợ
+ Quản lí đầu mối(Khách hàng/NXB)
